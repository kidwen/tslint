# tslint
tslint for your projects

## publish
```bash
$ npm publish
```

- publish to `https://npmjs.org`
- 不再发布到[github](https://npm.pkg.github.com)仓库

## install this package in your project

```bash
$ npm i @kidwen/tslint
```
